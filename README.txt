Radiant Glow Light V3

Deploy to Netlify:
1. Open this folder and make sure index.html is in the root.
2. Drag the full folder contents (not the parent folder) into Netlify Deploys.
3. After deploy, verify:
   - /
   - /collections.html
   - /inspiration.html
   - /sitemap.xml

Included:
- index.html
- collections.html
- inspiration.html
- about.html
- faq.html
- 404.html
- style.css
- assets/
- sitemap.xml
- robots.txt
- _redirects
- netlify.toml
